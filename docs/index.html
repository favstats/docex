One-minute demo

Respond to peer review in 3 commands:

# Replace a number (shows as tracked change in Word)
docex replace paper.docx "268,635" "300,000" --author "Fabio"

# Add a comment to a phrase
docex comment paper.docx "platform governance" "Added Suzor 2019" --by "Fabio"

# Format for Political Communication journal
docex style paper.docx --preset polcomm

Or use the JavaScript API:

const docex = require('./src/docex');
const doc = docex("paper.docx");
doc.author("Fabio Votta");

doc.replace("enforcement gap", "regulatory gap");
doc.at("platform governance").comment("Added Suzor 2019 citation");
doc.after("Methods").insert("We used a mixed-methods approach.");

await doc.save();

The .dex format

Every .docx becomes readable plain text. Every detail is preserved.

---
docex: "0.4.0"
authors:
  - name: "Un-named"
---

# QUARTERLY Strategic SYNERGY Report {id:BABCF972}

{p id:9E7DD00E}
{font "Lucida Handwriting"}{color 999999}Version 47.3b (DRAFT){/color}{/font}
{/p}

{comment id:0 by:"The Intern" date:"2024-05-13T22:00:00.000Z"}
WHO WROTE THIS???
{/comment}

Everything you need

Text editing

Replace, insert, delete, bold, italic, underline, highlight, color, strikethrough, superscript, subscript, small caps.

Comments

Add comments anchored to text, reply to existing comments, list and export all comments. Manages all 5 OOXML comment files.

Tracked changes

Every edit is tracked by default. Accept or reject changes individually or in bulk. Compare two document versions.

Figures and tables

Insert images with auto-dimensions. Create tables with booktabs styling. Bullet and numbered lists.

Journal styles

Built-in presets for Political Communication, APA 7, JCMC, JoC. Sets fonts, margins, spacing, and headers.

Academic tools

Word count, anonymize for blind review, Zotero citation injection, response letter generation, submission validation.
